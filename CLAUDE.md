# score-fetcher 프로젝트

## 프로젝트 개요
교회 예배 콘티(악보 세트리스트)를 관리하고, 악보 이미지를 자동으로 불러오는 단일 페이지 앱입니다.

## 주요 파일
- `index.html` — 앱 전체 (HTML + CSS + JS 단일 파일)
- `hymn_list.txt` — 찬양 목록 (번호 + 제목)
- `images/` — 악보 이미지 (001.png ~ ...)

## Firebase 구조
- Realtime Database URL: `https://score-fetcher-db-default-rtdb.firebaseio.com`
- `/history` — 콘티 이력 저장 (POST: 저장, DELETE /{key}: 삭제)
- `/fcm_tokens` — FCM 푸시 토큰 목록

## 주요 기능

### 악보 / 콘티 관리
- 찬양 번호 입력 → 악보 이미지 자동 표시
- 악보 전체화면 뷰어 (좌우 스와이프 이동, 풀블리드, pull-to-refresh 차단)
- 이력 불러오기 시 악보 초기화 + 첫 카드 자동 스크롤

### 콘티 이력
- Firebase `/history`에 저장 (저장 버튼 수동 저장, 악보 찾기 시 자동 저장 없음)
- 이력 모달: 불러오기(탭) / 삭제(길게 누르기, 700ms)
- 이력 검색 기능 (`renderHistoryList` 기반, 전체 캐시 후 필터)
- 콘티 순번 정확화 / 이중개행 제거

### 공유
- 콘티 이미지 생성 및 공유 (`shareConti`)
- 디자인: 어두운 단색 풀블리드 배경 (제목 해시 기반 8가지 팔레트 중 랜덤 선택, 콘티마다 일관)
- 날짜(`YYYY-MM-DD`) 부분 액센트 컬러 + 언더라인, 곡 번호 액센트 컬러
- 헤더(`| ... |`) 반투명 흰색, 코멘트(`-`) 살짝 작고 반투명
- Web Share API 지원, 폴백 시 이미지 다운로드

### 알림 / 설치
- FCM 푸시 알림 (Firebase SDK, VAPID 설정 완료)
- 앱 설치 버튼 (PWA, 오버레이 차단 대응, 미지원 기기 안내)
- 알림 활성화 버튼

### UX
- 화면 꺼짐 방지 (Wake Lock API)
- 카카오톡 인앱 브라우저 감지 → 외부 브라우저 유도 배너
- Open Graph 메타태그 (링크 미리보기)
- 모바일 터치 우선 UX
