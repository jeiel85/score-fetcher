# 🎵 찬양 악보 매니저 (Praise Sheet Music Manager)

> **찬양 콘티 리스트를 입력하면 자동으로 악보를 찾아주고, 모바일 환경에 최적화된 저장을 돕는 스마트 비서 앱입니다.**

이 프로젝트는 찬양팀 리더가 예배 준비 시간을 단축하고, 팀원들과 콘티 이력을 효율적으로 공유하기 위해 제작되었습니다.

---

## ✨ 핵심 기능 (Key Features)

### 🔍 스마트 악보 검색 및 정제
* **텍스트 자동 정제:** `1. 제목`, `02.제목`, ` 685 . 제목 ` 등 다양한 형식으로 입력해도 `001 제목` 형식으로 자동 변환합니다.
* **불필요한 개행 제거:** 복사/붙여넣기 시 발생하는 빈 줄을 자동으로 청소하여 깔끔한 리스트를 유지합니다.
* **멀티 확장자 탐색:** `jpg`, `png`, `gif`, `jfif` 등 다양한 이미지 포맷을 자동으로 탐색하여 악보를 불러옵니다.

### 💾 효율적인 악보 저장 (UX 최적화)
* **모바일 갤러리 맞춤 저장:** 다운로드 최신순으로 정렬되는 모바일 환경을 위해 **'역순 저장'** 기능을 지원합니다.
* **일괄 다운로드:** 여러 장의 악보를 시간차를 두고 순차적으로 저장하여 브라우저 차단을 우회합니다.
* **지정 파일명 규칙:** `날짜AM/PM_순번_곡번호.ext` 형식으로 저장되어 파일 정렬이 매우 간편합니다.

### ☁️ 실시간 이력 관리 (Firebase 연동)
* **초고속 DB:** Google Firebase(Realtime Database)를 연동하여 밀리초(ms) 단위의 빠른 속도로 이력을 저장하고 불러옵니다.
* **팀 공유:** 특정 기기가 아닌 서버에 저장되므로, 앱에 접속하는 모든 팀원이 동일한 지난 콘티 이력을 공유할 수 있습니다.
* **제목 지정:** 각 콘티에 제목(예: 2026-03-15 주일예배)을 부여하여 관리가 용이합니다.

### 📱 사용자 편의 기능 (Utility)
* **화면 꺼짐 방지 (Wake Lock):** 연주 중 화면이 꺼지지 않도록 '화면 켜짐 유지' 기능을 제공합니다. (기본값 ON)
* **외부 리스트 연동:** `hymn_list.txt` 파일을 통해 600곡 이상의 전체 곡 목록을 관리하고 실시간 검색/추가할 수 있습니다.
* **반응형 디자인:** PC 브라우저에서도 모바일 앱 화면처럼 보이도록 최적화된 레이아웃과 '맨 위로' 버튼을 제공합니다.

## 🛠 기술 스택 (Tech Stack)

* **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
* **Backend/DB:** Google Firebase Realtime Database
* **Deployment:** Vercel / GitHub Pages
* **Data Management:** Google Apps Script (초기 모델), Fetch API

---

## 🚀 시작하기 (Getting Started)

1. 이 저장소를 **Fork** 하거나 복사합니다.
2. `images/` 폴더 내에 곡 번호로 된 악보 이미지들을 업로드합니다. (예: `001.jpg`, `022.png`)
3. `hymn_list.txt` 파일에 전체 곡 리스트를 관리합니다.
4. `index.html` 내의 `FIREBASE_URL` 변수에 본인의 파이어베이스 실시간 DB 주소를 입력합니다.

## 📝 개발 노트 (Developer Notes)

* **미래의 나를 위한 디버깅:** 모든 주요 함수에는 `Input`, `Output`, `핵심 로직`, `왜 이렇게 짰는지`에 대한 상세한 주석이 포함되어 있습니다.
* **브라우저 정책 준수:** 사용자 경험을 위해 첫 터치 시 Wake Lock을 활성화하고, Visibility API를 통해 앱 복귀 시 상태를 자동 회복합니다.
