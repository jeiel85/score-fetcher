## Summary
- **#128: 콘티 관리 고도화 - 콘티 복제 기능**: 기존 콘티를 기반으로 새로운 콘티를 손쉽게 생성할 수 있는 복제 기능을 추가했습니다.
- **#129: 최근 사용한 곡 퀵 메뉴**: 사용자가 최근에 선택한 곡 10개를 별도의 필터 없이 검색창 상단에서 즉시 선택할 수 있도록 개선했습니다.
- **#135: 가사 데이터 태그 시각화**: `lyrics.json`의 태그 정보를 활용하여 가사 모달 내에 [1절], [후렴], [브릿지] 등 섹션 구분 레이블을 자동으로 표시합니다.

## Features
### #128 콘티 복제
- 콘티 이력 항목에 "📋 복사" 버튼 추가
- 제목에 자동으로 "(복사본)"이 붙으며 즉시 편집 모드로 진입

### #129 최근 사용 곡
- `localStorage`를 활용하여 최근 선택한 10곡의 이력을 기기별로 관리
- 검색창이 비어있을 때 우선적으로 노출하여 입력 시간 단축

### #135 가사 섹션 배지
- 가사 데이터의 메타정보를 분석하여 시각적 구조화 제공
- 복잡한 가사에서도 원하는 섹션을 빠르게 찾을 수 있도록 개선
