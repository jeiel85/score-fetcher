## Summary (관리 효율화 및 커스텀 기능)
- **Feature Toggle (기능 스위치) 시스템**: 신규 기능들을 관리자 페이지에서 개별적으로 켜고 끌 수 있는 제어 시스템을 구축했습니다.
- **#128: 스마트 콘티 복제**: 기존 콘티를 기반으로 새 콘티를 만들 수 있는 '📋 복사' 기능을 추가하여 반복적인 입력 시간을 단축했습니다.
- **#129: 최근 사용 곡 퀵 메뉴**: 사용자가 최근에 선택한 곡 10개를 검색창 상단에 버튼으로 제공하여 빠른 콘티 작성을 지원합니다.
- **#135: 가사 섹션 지능형 분류**: 가사 데이터의 메타태그를 분석하여 [1절], [후렴], [브릿지] 등 섹션 구분 배지를 자동으로 표시합니다.
- **관리자 모바일 UX 최적화**: 관리자 대시보드를 모바일에서도 편리하게 사용할 수 있도록 반응형 레이아웃(카드 뷰, 가로 스크롤 탭)을 적용했습니다.

## Features
### ⚙️ Feature Toggle
- Firebase 실시간 DB 연동을 통해 앱 재배포 없이 기능을 즉시 제어 가능합니다.

### 📱 Admin Mobile UX
- 테이블 형태의 데이터를 모바일에서는 카드 형태로 자동 변환하여 가독성을 극대화했습니다.

## Testing
- **기능 검증**: 관리자 페이지에서 설정 변경 시 메인 앱의 UI가 즉시 반영되는지 확인했습니다.
- **반응형 테스트**: 다양한 모바일 기기(iPhone, Android) 해상도에서 레이아웃 정상 작동 확인.
